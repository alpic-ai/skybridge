Brings flight search directly into Claude. The Kiwi connector supports one-way/round-trip, flexible dates, multiple passengers, all cabin classes. Works with Claude and Claude Desktop.

<img
  src="/images/showcase-kiwi.png"

/>

## What's interesting
- Native widget rendering inside the chat thread — no popouts, no modals.
- Tool calls and UI state stay in sync via Skybridge's typed bridge.
- Same code path runs in Claude today, ChatGPT when ready.
