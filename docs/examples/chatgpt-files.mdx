import { ChatExample } from "/components/chat-example.jsx";

Handles ChatGPT files end to end: takes a file attachment, compresses it server-side, stores the archive in Cloudflare R2, and hands back a presigned download link. Relies on the ChatGPT Apps SDK file APIs, so it runs in ChatGPT.
